# Meine User ID kopieren

Es kann sein, dass du bei der Kontaktaufnahme mit dem Support nach deiner User-ID gefragt wirst.

Führe dazu folgende Schritte aus:

<Steps>
  ### Einstellungen in Sharezone öffnen

### "Mein Konto" öffnen

### "User ID" anklicken

Deine User ID wird in die Zwischenablage kopiert.

</Steps>
