# Mein Konto mit einer E-Mail verknüpfen

Damit deine Daten immer und überall erreichbar sind, musst du deinen Account mit einer E-Mail verknüpfen.
Du provitierst dann von folgenden Dingen:

- Automatisches Backup
Weiterhin Zugriff auf die Daten bei Verlusst deines Gerätes
- Nutzung auf mehreren Geräten
Daten werden zwischen mehreren Geräten sind synchronisiert

<WarningDie Anmeldung ist aus datenschutzrechtlichen Gründen erst ab 16 Jahren erlaubt</Warning>

Wenn du eine E-Mail mit deinem Konto verknüpfen möchtest, kannst du dies über die Einstellungen machen.
Führe dazu folgende Schritte aus:

1. Einstellungen in Sharezone öffnen
2. "Mein Konto" öffnen
3. Auf "Anmeldemethode" drücken
4. Zwischen E-Mail/Apple/Google wählen
<Accordion title:"E-Mail">
- E-Mail: 1. Nicknamen eingeben
          2. E-Mail eingeben
          3. Passwort eingeben
          4. Mit "Verknüpfen" bestätigen
</Accordion>

<Accordion title:"Apple">
- Apple:  1. Auf "Apple" drücken
          2. Weiteren Schritten folgen
</Accordion>

<Accordion title:"Google">
- Goole:  1. Auf "Google" drücken
          2. Weiteren Schritten folgen
</Accordion>
