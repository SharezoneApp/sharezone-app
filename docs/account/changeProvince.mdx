# Meinen Bundesland ändern

Mithilfe des Bundeslandes können wir die restlichen Tage bis zu den nächsten Ferien berechnen.

Wenn du dein Bundesland in Sharezone ändern möchtest, kannst du dies über die Einstellungen machen.
Führe dazu folgende Schritte aus:

1. Einstellungen in Sharezone öffnen
2. "Mein Konto" öffnen
3. Auf "Bundesland" drücken
4. Bundesland auswählen
